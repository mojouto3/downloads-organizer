## [1.0.0] - 2026-05-25

### Added
- Initial release
- Automatic sorting into 9 categories: Images, Videos, Audio, Documents, Archives, Code, Installers, Fonts, Torrents
- Duplicate filename handling (appends _1, _2, etc.)
- NSIS installer with Desktop shortcut
- Add/Remove Programs entry for clean uninstall
- Color-coded PowerShell output showing results
- "Press Enter to close" so window stays open after running
